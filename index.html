    Intro

    With the advent of the 21st century, we have experienced the rapid growth of the Internet. Today, Internet speed is still a very important link in the Internet, whether it is using the free wireless network of the bus on the bus, or using the operatorâ€™s outdoor 5G(NR)/4G(LTE)/3G(HSPA)/2G(EDGE) network, internet speed is an important indicator, browsing social media; video call with family and friends; online games, all require excellent network quality.

    There are also many users who frequently visit Github. Their ISP may block Github. For example, SSL Handshake Failed, Github domain name is polluted by DNS (commonly raw.githubusercontent.com), Github CDN provider : Fastly CDN IP blocked by ISP. Various problems often cause some users to use VPN to bypass the problem of Github being blocked by ISP. Therefore, the purpose of this Speedtest site is to allow you to test the network quality from your network to the Github server.

    Content

    The test method used in this Speedtest is to let the browser download 10MB, 30MB, 50MB, 80MB, 100MB Bin files, users can choose to download one of 10MB, 30MB, 50MB, 80MB, 100MB, or download them at the same time.

    Therefore , users will be able to observe the speed and quality of your network connection to the Github server

    This site is fully powered by Github and Cloudflare
